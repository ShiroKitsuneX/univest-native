# Univest - Fluxo do App

## 1. Tela de Bem-vindo
- Slogan: "Seu portal inteligente para todos os momentos da sua jornada acadêmica"
- Botão: "Começar Jornada"

## 2. Tela de Interesses
Selecione uma ou mais áreas de interesse:
- Ensino Médio e Técnico
- Vestibulares e ENEM
- Graduação
- Mestrado
- Doutorado
- Pós-doutorado
- Pesquisadores

## 3. Tela de Perfil do Usuário
"O que melhor descreve você agora? (Você pode alterar depois)"

| Opção | Descrição |
|-------|-----------|
| Ensino Médio | Cursando o ensino médio regular |
| Ensino Médio Técnico | Ex: COTUCA, ETEC, SENAI, IFSP |
| Pré-vestibulando | Me preparando para vestibulares |
| Foco no ENEM | Estudando para o ENEM / SISU |
| Graduando | Cursando uma graduação |
| Pré-mestrado | Buscando uma vaga em mestrado |
| Mestrando | Cursando o mestrado |
| Pré-doutorado | Buscando uma vaga em doutorado |
| Doutorando | Cursando o doutorado |
| Pós-doutorando | Realizando pesquisa pós-doutoral |
| Educação Continuada | Cursos livres, MBA, Especializações |

---

## 4. Telas Específicas por Perfil

### 4.1 Pré-vestibulando / Foco ENEM
- Lista de cursos de graduação: Medicina, Biologia, Geografia, Engenharia, Direito, etc.
- Vestibulares: FUVEST, UNICAMP, ENEM, SISU, etc.
- Calendário de provas
- Notas de corte

### 4.2 Ensino Médio / Ensino Médio Técnico
- Escolas técnicas: COTUCA, ETEC, SENAI, IFSP
- Processos seletivos
- Cursos técnicos disponíveis

### 4.3 Graduando
-grade curricular
- Oportunidades: estágios, bolsas, programas de extensão
- Eventos acadêmicos

### 4.4 Pré-mestrado / Mestrando
- Programas de mestrado: pago, gratuito, com bolsa, sem bolsa
- Áreas de pesquisa
- Linhas de investigação
- Orientadores

### 4.5 Pré-doutorado / Doutorando / Pós-doutorando
- Programas de doutorado
- Bolsas de pesquisa (CNPq, FAPESP, etc.)
- Publicações
- Eventos acadêmicos

### 4.6 Educação Continuada
- Cursos livres
- MBA
- Especializações

## 5. Siga Universidades
"Você verá as novidades delas no seu feed"

Lista de universidades para seguir (filtrada pelo perfil do usuário):
- USP, UNICAMP, UNESP, UFSCar, etc.
- Instituições públicas e privadas

---

## 6. App Principal

### Tab 1: Feed
- Publicações das universidades seguidas
- Notícias acadêmicas
- Eventos
- Oportunidades (estágios, bolsas)
- Feed personalizado por área de interesse

### Tab 2: Explorar
- Buscar universidades
- Buscar cursos
- Buscar vestibulares
- Buscar eventos
- Ver listas: Notas de Corte, Ranking de Universidades
- Minhas universidades (seguidas)

### Tab 3: Notas
- Registrar notas (ENEM, vestibulares, notas universitárias)
- Histórico de notas
- Comparar notas com notas de corte
- Simular possibilidades de ingressão

### Tab 4: Perfil
- Dados pessoais
- Editar perfil
- Minhas metas/goals
- Configurações
- Descobrir novas áreas (recomendações)
- Sair
